## Getting started

To get started with the app clone the repo and run the following commands:

```
$ bundle install --without production
```

Create the database:

```
$ rails db:create
```

Migrate the database:

```
$ rails db:migrate
```

Create the fake users:

```
$ rails db:seed
```

Run the app in a local server:

```
$ rails server
```
